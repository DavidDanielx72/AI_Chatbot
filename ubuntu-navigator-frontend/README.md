# Ubuntu Navigator — Frontend

Client-side code:
- src/routes/ — page routes (__root, index, $threadId)
- src/components/ — UI components (chat, sidebar, age gate, location picker, ai-elements, ui primitives)
- src/hooks/ — React hooks
- src/lib/ — client-safe utilities
- src/assets/, src/styles.css, public/ — assets and global styles

Note: this is a TanStack Start project; the full app requires the backend.
Use the "full" zip to actually run it.
